2. CNNs, RNNs, Transformers & LLMs

July 13–15 — Prof. Tang

Overview

Coming into this module, you’ve already run an RNN and seen what a hidden state does (Week 1, Day 5). Now we go deeper: how CNNs extract spatial structure from images, why LSTMs solve the vanishing-gradient problem that plain RNNs hit, and how the attention mechanism at the heart of Transformers changed everything. Day 3 ends with hands-on use of a real pretrained large language model.

Learning Objectives

  • Explain convolution and pooling and train a CNN image classifier from scratch
  • Visualize learned convolutional filters and intermediate feature maps
  • Explain the vanishing gradient problem and how LSTM / GRU gates address it
  • Train an LSTM on a sequence task and compare it to the Week 1 plain RNN
  • Derive the self-attention formula and explain why it replaced recurrence for most sequence tasks
  • Load a pretrained transformer with Hugging Face and practice prompt engineering

Day-by-Day Activities

Day 1 (Jul 13) — Convolutional Neural Networks

How do machines “see” images? We go from a raw pixel grid to a classifier, then open the lid to see what each layer learned.

  • Build intuition: apply kernels by hand to a tiny image (edge detection, blur, sharpen)
  • Train a CNN on CIFAR-10; compare accuracy to a plain MLP on the same task
  • Visualize learned filters in the first conv layer and feature maps at each stage
  • Grad-CAM: highlight which image regions drove the prediction

Day 2 (Jul 14) — LSTMs & the Vanishing Gradient Problem

You ran an RNN in Week 1 — today we break it on a long sequence and fix it with an LSTM.

  • Demonstrate vanishing gradients: train a plain RNN to recall information from 50 steps ago; watch it fail
  • Introduce the LSTM cell: forget gate, input gate, output gate — draw the diagram and trace a single step
  • Train an LSTM on the same task and compare learning curves
  • Extension: GRU as a lighter alternative; compare parameter counts

Day 3 (Jul 15) — Transformers & LLMs

From attention to chatbots — hands-on the architecture that runs the AI world right now.

  • Derive scaled dot-product attention: Attention(Q,K,V) = softmax(QKᵀ/√d)V
  • Visualize multi-head attention weights on a sentence: which tokens attend to which?
  • Load a small pretrained transformer with Hugging Face (transformers library) and run text generation
  • Prompt engineering: zero-shot vs. few-shot, chain-of-thought, and how framing changes outputs
  • Discussion: what can — and can’t — today’s LLMs do reliably?
