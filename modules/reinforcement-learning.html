3. Reinforcement Learning

July 16–17 — Prof. Du

Overview

Reinforcement learning (RL) is how agents learn by trial and error — taking actions, receiving rewards, and improving their behavior over time. Over two days, students implement a tabular RL algorithm by hand, then see how neural networks let RL scale to richer environments.

Learning Objectives

  • Describe the agent-environment-reward loop and the idea of a Markov decision process
  • Implement tabular Q-learning and observe an agent learn a policy
  • Explain the exploration vs. exploitation tradeoff
  • Explain how a neural network can approximate Q-values (deep Q-learning)
  • Train and evaluate a deep RL agent on a classic control task

Day-by-Day Activities

Day 1 (Jul 16) — RL Fundamentals & Tabular Q-Learning

Learn the language of RL, then build an agent from scratch.

  • Agent, environment, state, action, reward — explored through an interactive grid-world game
  • Implement tabular Q-learning on FrozenLake (Gymnasium)
  • Visualize the learned Q-table and the agent's policy as it improves over episodes
  • Experiment with the exploration rate and observe its effect on learning speed

Day 2 (Jul 17) — Deep Reinforcement Learning

Scale up with neural networks.

  • From Q-tables to Q-networks: why function approximation is needed for larger state spaces
  • Train a Deep Q-Network (DQN) on CartPole (Gymnasium)
  • Plot reward-per-episode curves and discuss training stability
  • Wrap-up discussion: where else could RL be applied? (games, robotics, recommendation, traffic control)
