1. Foundations

July 5–10 — Ross Greer

Overview

This module builds the toolkit that every other module relies on. Students learn to wrangle data with NumPy and Pandas, train classic machine learning models with scikit-learn, build a simple neural network in a deep learning framework, and — just as importantly — visualize and interpret what their models are actually doing.

Learning Objectives

  • Load, clean, and manipulate datasets with NumPy and Pandas
  • Explore data visually using Matplotlib / Seaborn
  • Train and evaluate a classification model with scikit-learn
  • Train and evaluate a regression model with scikit-learn
  • Build and train a simple neural network with a deep learning framework (PyTorch)
  • Visualize and interpret model behavior: loss curves, decision boundaries, confusion matrices, and learned representations

Day-by-Day Activities

Day 1 (Jul 5) — Python for Data Science

Get hands-on with NumPy arrays and Pandas DataFrames using a real-world dataset (e.g., Palmer Penguins or California Housing).

  • Array operations, broadcasting, and vectorization in NumPy
  • Loading, filtering, grouping, and joining data in Pandas
  • Mini-challenge: answer a set of data questions using only vectorized operations (no loops!)

Day 2 (Jul 6) — Exploratory Data Analysis & Visualization

Turn raw data into insight with plots.

  • Histograms, scatter plots, and pair plots with Matplotlib/Seaborn
  • Spotting correlations, outliers, and class imbalance
  • Mini-challenge: produce a "data story" — three plots that reveal something interesting about a dataset

Day 3 (Jul 7) — Classification with scikit-learn

Train models that sort data into categories.

  • k-Nearest Neighbors, decision trees, and logistic regression on a labeled dataset (e.g., handwritten digits)
  • Train/test splits, accuracy, and confusion matrices
  • Mini-challenge: tune one hyperparameter and explain how it changes the confusion matrix

Day 4 (Jul 8) — Regression with scikit-learn

Train models that predict continuous values.

  • Linear regression and decision-tree regression on a real dataset (e.g., housing prices)
  • Evaluating with mean squared error and R²
  • Visualizing predicted vs. actual values and residuals

Day 5 (Jul 9) — Intro to a Deep Learning Framework

Build your first neural network from scratch using PyTorch.

  • Tensors, layers, loss functions, and optimizers
  • Train a small multilayer perceptron on an image dataset (e.g., MNIST or Fashion-MNIST)
  • Compare the neural network's accuracy to Day 3's classical models

Day 6 (Jul 10) — Visualizing Model Behavior & Showcase

Open the black box and share what you've built.

  • Plotting training/validation loss curves and diagnosing over- and under-fitting
  • Visualizing decision boundaries and learned feature representations (PCA / t-SNE)
  • Lightning talks: each student presents one plot that explains how their model "thinks"
